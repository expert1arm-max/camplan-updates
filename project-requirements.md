# Project Requirements

## Цель
Сделать Windows desktop-приложение для учёта камер на объектах: склад, офис, магазин и другие.

## MVP
- CRUD объектов.
- CRUD этажей и зон.
- Простой редактор плана.
- Камеры на плане с сектором обзора.
- Карточка камеры с копированием IP, RTSP и пароля.
- Поиск по названию, IP, объекту, зоне и статусу.
- Таблица всех камер.
- Локальное сохранение данных.
- Импорт/экспорт JSON.
- Экспорт камер в CSV.
- Demo data при первом запуске.

## Ограничения
- Не добавлять backend.
- Не подключать Supabase.
- Не делать пароль видимым по умолчанию.
- Не превращать редактор в CAD.
- Не переписывать Lovable UI с нуля.
