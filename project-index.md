# Project Index

## Что это
Desktop-приложение `CCTV Manager` для Windows на базе React/Vite/TanStack Router с Electron-оболочкой.

## Текущий статус
- Дизайн Lovable сохранён как основа.
- Добавлен Electron main/preload.
- Добавлено локальное хранение через IndexedDB.
- Подключены JSON/CSV импорт и экспорт.
- Сборка frontend и Windows packaging проходят.
- По умолчанию приложение открывается в безопасном режиме просмотра, редактор включается вручную.
- Инструмент `Кабель / Connector` работает как polyline с промежуточными точками и редактированием маршрута.
- Для плана добавлены цвета объектов, lock, outline-only rooms, hit areas и явный сброс selection.

## Основные экраны
- План объекта с объектами, зонами, планом и устройствами.
- Таблица всех устройств.
- Кабельные связи между устройствами на плане.
- Правая панель свойств устройства или элемента плана.
