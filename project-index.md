# Project Index

## Что это
Desktop-приложение `CCTV Manager` для Windows на базе React/Vite/TanStack Router с Electron-оболочкой.

## Текущий статус
- MVP desktop-версии реализован и упаковывается.
- Дизайн Lovable сохранен как основа, без переписывания UI с нуля.
- Добавлены Electron `main`/`preload` и безопасный IPC для файловых диалогов.
- Добавлено локальное хранение через IndexedDB.
- Актуальная модель данных использует `devices` и `deviceConnections`; legacy JSON с `cameras` нормализуется при импорте.
- Подключены JSON import/export, CSV export устройств и JPG export плана.
- По умолчанию приложение открывается в безопасном режиме `Просмотр`, режим `Редактирование` включается вручную.
- Инструмент `Кабель / Connector` работает как polyline с промежуточными точками, snap к устройствам и редактированием маршрута.
- Для плана добавлены цвета объектов, `locked`, outline-only rooms, hit areas и явный сброс selection.
- `Room` создается одним кликом как квадрат 200x200 по центру клика, берет preset color и после создания возвращает tool в `select`.
- `Door` теперь создается drag-to-draw жестом по стенам и границам комнат с определением стороны по направлению рисования и hover-preview.
- Добавлен инструмент `Полукруглая стена`: он создаёт `wall` с формой дуги, настраиваемым изгибом и общей логикой цвета/толщины стены.
- При рисовании обычных и полукруглых стен начало и конец линии магнитятся к существующим стенам и границам помещений.
- Выделенные стены показывают ручки на обоих концах: линию можно удлинять или укорачивать вдоль её направления, а при отпускании ручки или всей стены рядом с другой стеной/границей помещения конец автоматически подгоняется до стыка.
- Дефолтные цвета плана теперь стандартизированы: `room` и `wall` по умолчанию чёрные, `door` коричневая, в правой панели быстрые цвета ограничены шестью базовыми вариантами.
- Левая и правая панели работают как overlay drawers и не сдвигают canvas.
- Правая панель свойств открывается по selection, умеет pin и закрывается по клику в пустоту, если не закреплена.
- Для каждого этажа сохраняется свой viewport карты, а также состояние левой и правой панелей `collapsed/pinned`.
- `Ctrl+0` на карте центрирует активный этаж и подгоняет масштаб под все элементы плана.
- Верхняя левая подпись на карте показывает текущий объект и зону, выбранные в левом меню.
- В toolbar есть переключатель `Показать IP адреса`, который выводит IP-подписи на карте у устройств с заполненным IP.
- На старте `roomColorPreset` сбрасывается на чёрный, чтобы новые помещения не унаследовали старый выбранный цвет.
- Новые помещения теперь создаются чёрными независимо от выбранного preset color, а цвет в панели остаётся для ручной правки существующих элементов.
- В верхней панели `Файл`, `Редактирование` и `Все устройства` стоят в начале строки, рядом с заголовком приложения.
- Бейдж режима `Просмотр/Редактирование` закреплён в конце строки с фиксированной шириной, чтобы переключение режима не двигало остальные элементы тулбара.
- При завершении редактирования отдельное сообщение `Изменения сохранены` в верхней строке больше не показывается.
- Строка `Автосохранение • время` из верхней панели убрана.

## Основные экраны
- План объекта с объектами, зонами, элементами плана, устройствами и кабелями.
- Таблица `Все устройства` с поиском, фильтрами, сортировкой и CSV export.
- Правая панель свойств объекта, устройства, элемента плана или кабеля.
- Toolbar с режимом просмотра/редактирования, инструментами редактора, поиском и меню `Файл`.

- Инструмент `Text` теперь открывает inline-input прямо на карте вместо `prompt`, текст вводится на месте клика и фиксируется Enter/blur.
- Инструмент удаления убран из верхней панели; удаление осталось на объектах и через клавиши.
- Text tool uses an inline input on the canvas; empty blur no longer deletes the draft, preventing accidental loss while typing.
- Text draft focus now tracks the draft id only, so typing multiple characters does not reselect/overwrite the current input on every keystroke.
- Text draft shows an inline Enter hint beside the input so users know how to confirm canvas text.
- Text draft now commits when clicking back on the map, so a stray canvas click no longer starts a fresh text item.
- Text overlay no longer shows an Enter hint; text size is editable from the right panel.
- Text elements no longer expose manual width/height fields; their block size follows label text and font size.
