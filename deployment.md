# Deployment

## Web dev
```bash
npm run dev
```

## Desktop dev
```bash
npm run dev:desktop
```

## Frontend build
```bash
npm run build
```

## Windows package
```bash
npm run dist:win
```

## Portable EXE
```bash
npm run dist:portable
```

## Output
- Frontend build goes to `dist/`.
- Electron packaging output goes to `release/`.
- Portable build outputs a single `.exe` in `release/`.
