# CCTV Manager Updates

Official update repository for CCTV Manager.

CCTV Manager is a desktop application for designing, organizing, and managing CCTV systems and security layouts.

## Features

* CCTV floor planning
* Camera and device management
* Cable routing
* Multi-floor projects
* JSON project save/load
* JPG export
* CSV export
* Auto updates

## Platform

* Windows Desktop
* Electron-based application

## Releases

This repository is used to distribute official application updates.
